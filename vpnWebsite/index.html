Real nerd hours

  • What's with the name? Smoothwall is a linux distro used by schools to easily block certain ports and restrict access to certain sites and apps. Kings also uses this firewall method.
  • This VPN runs on TCP port 443 (also used by HTTPS) which can be used by VPNs since its SSL encrypted. Most/all VPNs on the appstore use UDP ports ~1195-7. UDP is usually faster (although less consistent), but the reason this VPN uses 443 is because the school blocks the aforementioned UDP ports, which is why most other VPNs won't work on the school wifi. Why doesn't school also block 443? Well as mentioned this is the port through which all HTTPS requests are sent through so they need it open. Those other UDP ports are basically reserved for VPNs so the school has no incentive to keep them open.
